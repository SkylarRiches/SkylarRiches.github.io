About Me

I am a computer games programming student at the University Of Gloucestershire. I am particularly enthusiastic about creating games using the Unity game engine.

I am focused on gameplay programming for games built using the Unity Engine, but I also enjoy designing and building levels. I have experience in working solo, as a team member and as a team lead.

Unity Projects

Pacman Remake

As part of one of my level 4 projects, I had to recreate a classic retro game of my choosing. I chose to make pacman as it was one my lecturer classified as high complexity due to the collision system needed.

All of the code I wrote was written in C++ using Visual Studio.

Libraries used: SDL 2.0

OBJ Model Render Viewport

As part of one of my level 5 projects, we were taught how to make a 3D model render viewport using OpenGL. The render viewport has the capability to load an OBJ file and then apply a texture to it using the mtl and tga files.

All of the code I wrote was written in C++ using Visual Studio.

Libraries used: GLAD, GLFW, GLM, IMGUI, STB
