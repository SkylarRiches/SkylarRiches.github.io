About Me

I am a computer games programming student at the University Of Gloucestershire. I am particularly enthusiastic about creating games using the Unity game engine.

I am focused on gameplay programming for games built using the Unity Engine, but I also enjoy designing and building levels. I have experience in working solo, as a team member and as a team lead.

Unity Projects

Advanced First Person Controller

Due to my love for the game 'Titanfall 2' I decided to try and recreate the first person character controller from that game as a personal project. The controller was built using a rigidbody instead of a character controller.

Features of the character controller: Basic Movement (including sprint, crouch and jump), Crouch Slide, Wall Run, Slope Sliding and Smooth Terrain Movement.


Tools used: Unity, Unity Input System, Probuilder, C#

Procedural Solar System w/ Terrained Planets

For one of my projects at level 5 in university, I made a procedural solar system where there is a central star that planets orbit, and with the click of a button you can spawn in a new planet to orbit the star.

Each planet created has simple procedural terrain around its surface using simplex perlin noise. The coloration of the planet seen from orbit is determined by how close it is to the star, icey blue when far, hot red when close.


Tools used: Unity, C#

Cook It: A Cooking Game Played With A BopIt

At level 5 in university, we had a group assignment where we worked with Game Design students to make a game that was experimental in some way, be it through gameplay or control scheme.

For the assignment, my team modified a standard Bop It to work as a controller for a cooking game. Each control on the Bop It controlled a differerent cooking function. The gameplay itself is quite simple as you go through the steps in a linear order, however I plan to rework the game to have more fun gameplay.

My role for the project was team lead and lead programmer.

Tools used: Unity, C#
